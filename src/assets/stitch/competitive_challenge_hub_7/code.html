Calculate the value of x in the following geometric sequence where the sum of the first three terms is 42 and the common ratio is 2.

architecture

a + ar + ar² = 42

r = 2

check_circle Correct Answer

x = 6

Explanation

Using the formula for a geometric sequence sum: a(1-rⁿ)/(1-r). For three terms: a + 2a + 4a = 42. Therefore, 7a = 42, which results in a = 6.
