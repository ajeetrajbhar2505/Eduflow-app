A company is designing a new cloud architecture. They need to ensure high availability and low latency for users globally. Which of the following strategies would be most effective for achieving this goal?

Your Answer

close

Deploying all resources in a single region to minimize internal data transfer costs.

Correct Answer

check

Implementing a multi-region deployment with a global content delivery network (CDN).

lightbulb

Explanation

High availability is best achieved by distributing workloads across multiple geographic regions to protect against regional outages.

Low latency for global users is primarily addressed by using a Content Delivery Network (CDN), which caches content at edge locations closer to the end-users. Single-region deployments (Option A) are susceptible to total downtime if that region fails and increase latency for distant users.
